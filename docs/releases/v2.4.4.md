# Go v2.4.4

v2.4.4 is a bot-focused permissions and admin usability release. It adds explicit broadcast-group ownership for primary global operators and improves the private-chat cleanup editor on mobile. The release workflow republishes both versioned images, but no chain-watcher business logic changed from v2.4.3.

## Broadcast group ownership

- Adds `broadcast_groups.owner_user_id`, restricted by PostgreSQL constraints and triggers to primary global operators.
- Lets the host manage every group while each primary manages only groups it owns.
- Keeps explicit group-use permission separate from ownership and management rights.
- Requires an exact direct-chat permission before a primary can add a chat or delegate it independently.
- Prevents a primary from managing another primary's secondary operators or revoking grants created by someone else.
- Adds append-only broadcast-group audit events and a repair-candidate table for ambiguous historical ownership.
- Records the forward-only migration gate `2.4.4-broadcast-group-ownership`.

## Admin cleanup settings

- Moves private-chat cleanup settings into a focused modal instead of expanding the operator list.
- Supports independent hour and minute inputs while preserving the existing cleanup scheduler behavior.
- Adds a compact mobile search field for filtering operators without exposing unrelated identities.
