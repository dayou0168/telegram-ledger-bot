# Go v2.4.11

v2.4.11 is a bot-only broadcast replacement correctness release. It preserves the original photo caption when an administrator configures a replacement image without configuring fixed replacement text.

## Fixed

- A single-chat broadcast containing a photo and caption no longer loses its caption when the replacement setting contains only an image.
- Fixed replacement text still overrides the original caption when it is non-empty.
- The in-place media edit path and the send-new-photo/delete-old-message fallback now use the same caption rule.
- Empty replacement settings still remain a no-op and are not marked as completed replacements.

## Compatibility

- No PostgreSQL migration is added.
- No watcher protocol, scheduler, database, or chain source behavior changes.
- Existing broadcast replacement settings remain compatible.
- Production may update bot containers only; `ledger-chain-watcher` does not need to restart.

## Verification

- `go test ./internal/bot`
- `go test ./...`
- PostgreSQL CI and critical race tests in GitHub Actions
